#### <span data-lang="en">B.1: Create a simple C++ application</span><span data-lang="fr">B.1: Créer une application C++ simple</span>

<span data-lang="en">Prompt to generate C++ code:</span><span data-lang="fr">Prompt pour générer du code C++ :</span>
```
You are an experienced C++ developer.
Create a simple program that:
- Reads the user's name and age
- Calculates how many years until the user turns 100
- Displays a message with the result
The code should be well-commented to facilitate understanding.
Create a .md file with all information for setup and execution.
```

#### <span data-lang="en">B.2: Convert the C++ application to Python</span><span data-lang="fr">B.2: Convertir l'application C++ en Python</span>

<span data-lang="en">Prompt for conversion:</span><span data-lang="fr">Prompt pour la conversion :</span>
```
You are a system modernization expert.
Convert the following C++ code to Python, keeping the original logic but applying modern best practices:
- Use of functions
- Input validation
- Clear messages
Explain the main changes made.
```
