# Ajouter les heures travaillées pour un employé
**ID:** 3204  
**Type:** User Story  
**Status:** New  
**Priority:** 2  

## Description
En tant que gestionnaire RH,
Je veux pouvoir saisir les heures de travail quotidiennes pour chaque employé,
Afin de calculer correctement le total des heures et préparer la paie.

## Acceptance Criteria
- Je peux sélectionner un employé dans la liste.
- Je peux entrer la date, l'heure de début et l'heure de fin.
- Le système valide que l'heure de fin est après l'heure de début.
- Un message d'erreur s'affiche si les heures dépassent 24 heures par jour.
- Les données sont sauvegardées et visibles dans le rapport hebdomadaire.
