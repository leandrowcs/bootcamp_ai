# <span data-lang="en">Add Worked Hours for Employee</span><span data-lang="fr">Ajouter les heures travaillées pour un employé</span>
**ID:** 3204  
**<span data-lang="en">Type</span><span data-lang="fr">Type</span>:** User Story  
**Status:** New  
**<span data-lang="en">Priority</span><span data-lang="fr">Priorité</span>:** 2  

## <span data-lang="en">Description</span><span data-lang="fr">Description</span>

<span data-lang="en">
As an HR manager,
I want to be able to enter daily work hours for each employee,
So that I can correctly calculate total hours and prepare payroll.
</span>

<span data-lang="fr">
En tant que gestionnaire RH,
Je veux pouvoir saisir les heures de travail quotidiennes pour chaque employé,
Afin de calculer correctement le total des heures et préparer la paie.
</span>

## <span data-lang="en">Acceptance Criteria</span><span data-lang="fr">Critères d'acceptation</span>

<span data-lang="en">
- I can select an employee from the list.
- I can enter the date, start time, and end time.
- The system validates that the end time is after the start time.
- An error message appears if hours exceed 24 hours per day.
- Data is saved and visible in the weekly report.
</span>

<span data-lang="fr">
- Je peux sélectionner un employé dans la liste.
- Je peux entrer la date, l'heure de début et l'heure de fin.
- Le système valide que l'heure de fin est après l'heure de début.
- Un message d'erreur s'affiche si les heures dépassent 24 heures par jour.
- Les données sont sauvegardées et visibles dans le rapport hebdomadaire.
</span>
